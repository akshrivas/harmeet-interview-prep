# Interview War Room — Week 1

Senior frontend interview command center (vanilla HTML/CSS/JS, offline-capable).

## Live app

After GitHub Pages deploy: **https://akshrivas.github.io/harmeet-interview-prep/**

## Local use

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
# http://localhost:8080
```

## Pages

- **Command Center** — `dashboard.html`
- **Day drill** — `day.html?day=1` … `day=7`
- **Mock round** — `mock.html`

Progress is stored in the browser (`localStorage`).
